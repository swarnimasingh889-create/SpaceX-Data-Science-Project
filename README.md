# 🚀 SpaceX Data Science Project



A complete data science project including data collection, analysis, machine learning, and dashboard visualization.



## 📌 Project Overview

This project analyzes SpaceX launch data and builds machine learning models to predict launch success.



## 🔄 Project Steps

- Data Collection (API & Web Scraping)

- Data Wrangling

- Exploratory Data Analysis (SQL & Visualization)

- Machine Learning Models

- Dashboard using Plotly Dash



## 🛠️ Tools Used

- Python

- Pandas

- SQL

- Scikit-learn

- Plotly Dash
